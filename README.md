# Claude Skills

Small, focused skills for Claude Code. Each one ships with a stated
purpose, a real example, and — where it applies — a pre-declared
kill criterion I test against before calling it "done."

## Skills

### [engineering-historian](./engineering-historian)
Captures a chronological, per-project engineering record — decisions
made (and alternatives rejected), verified commands, mistakes, open
questions. The principle: *the log is the asset, the reasoning is
rented.* Currently in a 60-day usage trial.

More skills landing here over time.

## Install
Copy a skill's folder into `.claude/skills/<name>/` in your project
or `~/.claude/skills/<name>/` globally.
